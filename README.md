# Intel 8085 Inspired Microprocessor in Verilog HDL

RTL implementation of the **Intel 8085 microprocessor** in Verilog HDL. The processor is designed from scratch consisting of several modules like Control Unit, Arithmetic Logic Unit, Program Counter, Instruction Decoder, Register file etc. The execution of instruction is driven by a Moore FSM (Finite State Machine).

**Status:** 🚧 Work in Progress (Core datapath completed, instruction set under development)

# Instructions Supported
| Category | Instructions |
|----------|--------------|
| Data Transfer | MOV r,r  MVI r,data  LXI rp,data 16 |
| Arithmetic | ADD r |

## Features
- Moore FSM based Timing & Control Unit
- 8-bit ALU
- Register File (A,B,C,D,E,H,L)
- Program Counter
- Stack Pointer
- Temporary Register
- WZ Register
- Instruction Register
- Instruction Decoder
- Internal Address Bus Multiplexer
- Internal Data Bus Multiplexer
- Flag Register with Selective Flag Update (Using Flag mask generated by the Control Unit)
